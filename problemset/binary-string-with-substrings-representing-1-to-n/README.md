# 子串能表示从 1 到 N 数字的二进制串

> 难度：中等
>
> https://leetcode.cn/problems/binary-string-with-substrings-representing-1-to-n/

## 题目

给定一个二进制字符串 `s` 和一个正整数 `n`，如果对于 `[1, n]` 范围内的每个整数，其二进制表示都是 `s` 的 子字符串 ，就返回 `true`，否则返回 `false` 。

子字符串 是字符串中连续的字符序列。

 

### 示例 1：
```
输入：s = "0110", n = 3
输出：true
```

### 示例 2：
```
输入：s = "0110", n = 4
输出：false
```
